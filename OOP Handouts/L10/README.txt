=====
SQL files should be transferred to your Stanford UNIX directory
and loaded into the MySQL database using the SOURCE command.

Don't forget to edit the USE statement in these files to load 
into your database, not mine.

metropolises.sql -- metropolises example from MySQL handout
universities.sql -- universities example used to JOIN with metropolises in
   MySQL handout
   
=====
ExampleAccess.java

This file demonstrates accessing MySQL from Java.  Before running
this, you'll need to download the MySQL/JDBC driver as discussed
in the JDBC handout.  In addition you'll need to edit the file to
use your account name, password, and database name.
